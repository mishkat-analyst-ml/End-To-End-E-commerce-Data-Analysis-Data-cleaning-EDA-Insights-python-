## End-To-End-E-commerce-Data-Analysis-Data-cleaning-EDA-Insights-python

A complete data analysis project on Pakistan’s e-commerce sales dataset. It covers data cleaning, exploratory data analysis (EDA), and business insights extraction using Python. The project helps identify sales trends, customer patterns, and key business opportunities through data visualization.


## Data Cleaning Process

In this project, I performed the following data cleaning steps:

First i Removed unnecessary columns that were not useful for analysis

Then Handled missing values by filling or removing null entries

Counted null values in each column to understand data quality

Converted year and month columns from float datatype into proper DATE format

Created a new column called Revenue for sales analysis

📊 Exploratory Data Analysis (EDA)

## After cleaning the dataset, I performed EDA to understand patterns and trends:

Analyzed top products by revenue

Studied sales distribution across different categories

Identified yearly sales performance trends

Compared payment methods based on revenue contribution

Analyzed discount distribution across product categories
